# InventoryApp
This is an inventory application for e-commerce.
- Spring Boot
- Spring Data Jpa
- Thymelaf
- MySQL

## Requirements
- JDK 1.8 or newer
- Maven 3
- Lombok

## Port
http://localhost:2323
